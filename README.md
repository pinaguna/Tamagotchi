# Lokumotchi -  Virtual Pet Game

A Tamagotchi-like game built with Unity.
The game runs in a small window and features a virtual cat with three status bars: hygiene, happiness, and hunger.
Interacting with different objects in the room increases or decreases these values, encouraging continuous care and interaction.

## Features

-Virtual pet

-Small desktop window gameplay

-Three status bars: cleanliness, happiness, and hunger

-Interactive room objects affecting pet stats
